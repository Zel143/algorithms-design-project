## Working Rules

- Keep the solution aligned with the handout requirements.
- Prefer algorithm clarity, complexity analysis, and test coverage over extra features.
- Use the topic areas from the handout as the project scope.
- Keep all work original and cite sources using ACM style.
- Preserve a small, clear workflow instead of adding unnecessary structure.

## Logging

Record each major AI action in [ai_workflow_logs.md](ai_workflow_logs.md).

Each log entry should include:

- Date and time
- Task or decision made
- Files changed
- Validation performed
- Remaining follow-up items

## Default Order

1. Review the rules file.

3. Map the work to the project handout requirements.
4. Make the smallest useful edit.
5. Validate the change.
6. Log the result.
