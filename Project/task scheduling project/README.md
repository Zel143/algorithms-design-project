# Task Scheduling Project

This folder is organized in the order requested for the project demo:
1. Problem
2. Code
3. Output
4. Report

## Problem
Task Scheduling with Deadlines and Profits.

Given a set of tasks each with a deadline and a profit, schedule them to
maximize total profit. Each task takes one time unit and must be completed
on or before its deadline.
